# Product Overview

AWS 2-tier web infrastructure with automated CI/CD pipeline for AWS Migration Services Competency compliance (MOB-005 Operations).

## Core Components

- VPC with multi-AZ public/private subnets
- Application Load Balancer for high availability
- EC2 instances running Nginx web servers
- Automated deployment via CodePipeline and CodeBuild
- Multi-environment support (dev/prod)

## Compliance Focus

Implements AWS Migration Services Competency requirement MOB-005 (Operations) with:
- Infrastructure-as-code using CloudFormation
- Automated deployment and testing
- Operational monitoring and alerting
- Runbooks and SOPs for operations management
